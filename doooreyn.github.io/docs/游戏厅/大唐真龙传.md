# 大唐真龙传

## 一、游戏介绍

> 【简单介绍】大唐真龙传，**首款隋唐题材手游回合制策略卡牌 RPG**，Q 版酷炫美术风格。丰富的职业系统，独创的回合奖励。带你体验与众不同的回合制策略 RPG!
>
> 【游戏特色】秦琼，程咬金等传奇名将带你领略英雄辈出的隋唐乱世!先锋，谋士，射手，猛将，将帅。不同职业不同的特点，带你领略不一样的 RPG 战斗!率领众将挥师东瀛，占领全岛获得丰厚奖励。宝玉，完美，武将，美女，统统全撸走!
